The Best Magnetic Accessories
寻找最好的磁吸配件

MagSnap is building an ecosystem, not just one product.

We are looking for the world’s best magnetic accessories that can integrate into the MagSnap Ecosystem.

This includes, but is not limited to:

  • Headlamps / Action Cameras / DJI Mic
    头灯 / 运动相机 / DJI Mic
  • AirTag / Radios / Flashlights
    AirTag / 对讲机 / 手电
  • Safety Lights / Outdoor Tools / Future Magnetic Modules
    安全警示灯 / 户外工具 / 未来磁吸模块

MagSnap 正在打造一个磁吸生态，而不仅仅是一副眼镜。

我们正在寻找全球最好的磁吸配件，并将它们整合到 MagSnap Ecosystem。

包括但不限于：

  • 头灯 / 运动相机 / DJI Mic
  • AirTag / 对讲机 / 手电
  • 安全警示灯 / 户外工具 / 未来磁吸模块

真实用户反馈：

  • 当前测试中发现：部分头灯磁铁胶水容易脱落。
  • 当前测试中发现：1mm N52 小磁铁容易碎裂。

因此我们不仅寻找更好的头灯，更寻找整个磁吸生态中更优秀的产品和合作伙伴。

The goal is not to manufacture every product ourselves. The goal is to find the world’s best products and build the MagSnap Ecosystem.
